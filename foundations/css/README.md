# CSS Foundations

Goals:
- Learn selectors, cascade, and specificity.
- Practice layout with Flexbox and Grid.
- Build responsive designs with fluid units and media queries.
- Understand CSS variables and modern styling techniques.

Platform Alignment:
- FreeCodeCamp: Responsive Web Design Certification
- Scrimba: CSS Fundamentals and Responsive Design
- Boot.dev: CSS basics for web development

Try:
- Style the HTML foundations page using `styles.css`.
- Create a two-column layout and make it stack on small screens.
- Implement a responsive navigation menu.
- Use CSS variables for consistent theming.
- Create custom properties and apply advanced selectors.
